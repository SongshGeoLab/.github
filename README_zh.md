## 关于

**宋爽的虚拟实验室（SongshGeo Lab）** 是一个跨学科研究团队，研究方向包括：
- 社会-生态系统
- 计算地理与建模
- 社会水文学与水资源管理
- 历史地理与制度分析

由宋爽博士领导，研究基地位于马克斯·普朗克地球人类学研究所。
